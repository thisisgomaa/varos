# أمر شغل ٢ — تصفية باقي دفتر الأوجاع (تشغيل مستقل بكوديكس + مراجعة تخطيط)
**التاريخ:** 2026-07-08 · المصدر: `PAINS_LOG.md` + توضيحات أحمد 07-08 · **المنفّذ:** كوديكس تاسك-تاسك، التخطيط يراجع كل تاسك (بوابات + مراجعة عدائية للخطر) قبل الدمج والانتقال.

> **مانديت أحمد:** "اشتغل لوحدك، تاسك-تاسك، راجع كويس، مترجعش غير لما تخلص كله." كل تاسك: worktree معزول → كوديكس → مراجعة تخطيط + بوابات → دمج main → التالي. مفيش استنّي.

## توضيحات أحمد المثبّتة (مهمة — دقيقة)

### A30 — قفل العنصر على البورد (الشكل النهائي)
- **الكليب (لما البورد عليه clip=on) يقص كل العناصر** فلا يطلع أي عنصر برّا حدود البورد نهائيًا (على مستوى الرندر — العنصر يتقص بصريًا عند حد الصفحة).
- **إعفاء فردي:** كليك يمين على عنصر → خيار **"Release clip / اطلع برّا البورد"** → العنصر ده بالذات يتجاهل قص البورد ويطلع برّا عادي (يلغي الكليب فرديًا ليه هو بس). خاصية على العنصر (`clip_exempt` بـserde default false).
- القبول: بورد clip=on + ٣ عناصر بعضها يتعدى الحد → كلهم متقصوصين؛ كليك يمين على واحد → Release → يبان كامل بره الحد؛ الباقي متقصوص.

### P6 — إعادة تصميم رأس البوكس (سلوك Illustrator)
1. **شيل زرار الـ× خالص** من رأس البوكس.
2. **كليك يمين على البانل (التاب/الرأس)** → قائمة سياق زي Illustrator: **Close · Close Tab Group · (Minimize لاحقًا)**. Close يقفل البانل النشط؛ Close Tab Group يقفل البوكس كله بتابّاته.
3. **زرار الـ☰ الحالي يتغيّر معناه:** يبقى **إعدادات البانل نفسها** (panel flyout الخاص بالبانل — أوبشنز مستقبلية للبانل)، مش قائمة "غيّر البانل".
4. **قائمة "أنهي بانل تحط في البوكس" تتنقل للكليك يمين على المقابض:**
   - كليك يمين على **شريط التحريك (الجريب)** → قائمة البانلات → تختار واحدة **تنزل في البوكس ده**.
   - كليك يمين على **شريط التحجيم بين بانلين** → القائمة → البانل المختارة **تنزل بينهم** (بوكس جديد في النص).
- ده كله شغل shell/boxtree.rs — أخطر تاسك (تصميم)، يتاخد بعناية وبمراجعة أعلى.
