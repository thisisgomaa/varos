# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Short version: be kind, be professional, critique code not people, and remember there are humans — often volunteers — on the other side of every issue and PR. Harassment, discrimination, and bad-faith behavior are not tolerated, in any language.
